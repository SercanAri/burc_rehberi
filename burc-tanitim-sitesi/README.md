# Burç Tanıtım Sitesi

Modern, tek sayfalık ve statik bir burç tanıtım sitesi.

## İçerik

- 12 burç için tarih aralığı, element, kısa açıklama ve güçlü yönler
- Elementlere göre filtreleme
- Seçilen burç için detay paneli
- Responsive tasarım

## Çalıştırma

`index.html` dosyasını tarayıcıda açmanız yeterli.

GitHub Pages için bu klasörü bir depoya ekleyip Pages kaynağı olarak yayınlayabilirsiniz.
